# Cops and robbers

Cops and robbers is a two player game which takes place on a graph. Each turn the cop tries to catch up with the robber, until the robber is cornered.

## Drunk cops and (sober) robbers

In this varient of the game the cop moves completely randomly.
[Drunk cops](https://drunkcops.com) is a tool to explore, simulate, and visualize the game of cops and robbers (with plans to support more varients).

## Getting Started

### Installation

Install the dependencies:

```bash
npm install
```

### Development

Start the development server with HMR:

```bash
npm run dev
```

Your application will be available at `http://localhost:5173`.
